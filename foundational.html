Front-end Foundational Manual

background:

The `background` property in CSS is used to set the background color or image of an element. You can use various values like color names, RGB, RGBA, HEX, HSL, and images.

demo / example:

This is a background color example.

example usage:

background: rgba(112, 255, 255, 0.8);
background: #007722;

code in context:

.bg-color-demo {
  background: rgba(255, 255, 255, 0.8);
  padding: 20px;
}

padding:

The `padding` property in CSS is used to generate space around an element's content, inside of any defined borders. It can be set using length values, percentages, or the `auto` keyword.

demo / example:

z-index:

The `z-index` property in CSS is used to specify the stack order of an element. An element with a higher stack order is always in front of an element with a lower stack order.

Z-index 2

example usage:

z-index: 1;
z-index: 2;

code in context:

  padding: 10px;

hover effects:

The `:hover` pseudo-class in CSS is used to apply styles to an element when the mouse hovers over it. It can be used to change colors, add borders, and create animations.

demo / example:

example usage:

  color: white;
}

code in context:

}

  background: rgba(255, 112, 112, 0.8);
  color: white;
}

media queries:

The `@media` rule in CSS is used to apply styles based on the result of one or more media queries. It allows you to create responsive designs that adapt to different screen sizes and devices.

demo / example:

Resize the window to see the effect!

example usage:

@media (max-width: 600px) {
  .media-query-demo {
    background: rgba(255, 112, 112, 0.8);
    color: white;
  }
}

code in context:

.media-query-demo {
  background: rgba(112, 255, 255, 0.8);
  padding: 10px;
  width: 100%;
}

@media (max-width: 600px) {
  .media-query-demo {
    background: rgba(255, 112, 112, 0.8);
    color: white;
  }
